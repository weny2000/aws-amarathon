# 新版设计系统说明

## 设计理念

新版网站采用现代化的设计系统，参照旧版Amarathon网站的视觉风格，使用渐变紫粉色系作为主色调，打造专业、现代、充满活力的视觉体验。

## 主要特点

### 1. 色彩系统

#### 主色调 - 渐变紫粉色
- **主渐变**: `linear-gradient(98deg, #2d62ff -24.99%, #be52ff 107.17%)`
- **主蓝色**: `#2d62ff`
- **主紫色**: `#be52ff`
- **主粉色**: `#ec4899`

#### AWS 品牌色
- **橙色**: `#FF9900`
- **深色**: `#232F3E`

#### 文字颜色
- **主文字**: `#232f3f`
- **次要文字**: `#545e6b`
- **第三级文字**: `#556e8f`
- **浅色文字**: `#acb1b7`

### 2. 设计元素

#### 卡片设计
- 白色背景
- 圆角: 8px - 16px
- 阴影: `0 2px 12px rgba(0, 0, 0, 0.08)`
- 悬停效果: 上移 + 阴影加深

#### 渐变卡片
- 使用主渐变作为外边框
- 内部白色内容区
- 适用于: 演讲议程、组委会卡片

#### 按钮样式
- 主按钮: 渐变背景
- 次要按钮: 白色背景 + 渐变边框
- 圆角: 8px 或完全圆角
- 悬停: 轻微上移 + 阴影

### 3. 布局系统

### 5. 响应式设计

#### 断点
- 桌面: > 1199px
- 平板: 768px - 1199px
- 手机: < 768px

#### 适配策略
- 桌面: 多列布局
- 平板: 减少列数，调整间距
- 手机: 单列布局，优化触摸交互

### 6. 动画效果

#### 过渡动画
- 悬停效果: 0.3s ease
- 页面切换: 淡入 + 上移
- 按钮交互: 上移 + 阴影

#### 关键帧动画
- fadeIn: 淡入效果
- fadeInUp: 淡入上移
- spin: 旋转（加载状态）
- pulse: 脉冲（强调元素）

### 7. 无障碍设计

#### 焦点样式
- 蓝色外边框
- 2px 宽度
- 2px 偏移
- 半透明阴影

#### 键盘导航
- 所有交互元素可通过Tab访问
- 清晰的焦点指示
- 逻辑的Tab顺序

### 8. 特殊组件

## 使用指南

### 添加新组件

1. 遵循现有的命名规范
2. 使用CSS变量定义颜色和间距
3. 添加悬停和焦点状态
4. 确保响应式适配
5. 考虑无障碍需求

### 自定义颜色

修改 `:root` 中的CSS变量：

```css
:root {
    --primary-blue: #your-color;
    --primary-purple: #your-color;
    /* ... */
}
```

### 自定义间距

修改间距变量：

```css
:root {
    --spacing-md: 20px; /* 默认 24px */
    /* ... */
}
```
