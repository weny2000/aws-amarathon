## 使用方法

### 1. 查看效果

直接在浏览器中打开 `index.html` 即可看到新样式。

### 2. 自定义颜色

修改 `styles.css` 中的CSS变量：

```css
:root {
    --primary-blue: #your-color;
    --primary-purple: #your-color;
    /* ... */
}
```

### 3. 调整间距

修改间距变量：

```css
:root {
    --spacing-md: 20px;
    /* ... */
}
```

### 4. 恢复旧样式

如果需要恢复旧样式：

```bash
Copy-Item styles.css.backup styles.css
```
