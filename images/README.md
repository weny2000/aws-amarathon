# 图片资源说明

## 目录结构

```
images/
├── 2024/          # 2024年活动图片
│   ├── banner.jpg        # 活动横幅 (建议尺寸: 1200x400)
│   ├── photo1.jpg        # 活动照片 (建议尺寸: 800x600)
│   ├── photo2.jpg
│   └── ...
└── 2025/          # 2025年活动图片
    ├── banner.jpg
    ├── photo1.jpg
    └── ...
```

## 图片规格建议

### 横幅图片 (Banner)
- 尺寸：1200x400 像素
- 格式：JPG 或 PNG
- 文件大小：< 500KB

### 活动照片 (Gallery)
- 尺寸：800x600 像素（4:3比例）
- 格式：JPG 或 PNG
- 文件大小：< 300KB

## 使用方法

1. 将图片文件放入对应年份的目录
2. 在 `data.json` 中更新图片路径

示例：
```json
{
  "2024": {
    "banner": {
      "image": "images/2024/banner.jpg",
      "alt": "Amarathon 2024"
    },
    "gallery": [
      {
        "image": "images/2024/photo1.jpg",
        "caption": "开幕式现场"
      }
    ]
  }
}
```

## 注意事项

- 图片文件名使用英文和数字，避免使用中文
- 保持图片文件大小合理，优化网站加载速度
- 如果没有图片，留空 `image` 字段，系统会显示占位符
